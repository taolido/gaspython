## xlsxファイルの取得方法（推奨）

### Google Colabで生成（最も簡単）

1. [Google Colab](https://colab.research.google.com/) で新しいノートブックを開く
2. `generate_xlsx_colab.py` の内容をセルにコピペ
3. 実行（Shift+Enter）
4. xlsxファイルが自動ダウンロードされる

```python
# Colabで実行すると、以下のファイルがダウンロードされます
# - sales_data_202401.xlsx
# - sales_data_202402.xlsx
# - sales_data_202403.xlsx
# - employee_list.xlsx
# - product_master.xlsx
# - sample_data.zip (merge_sample/の10ファイル)
```

## 同梱ファイル

### CSVファイル（そのまま使える）

| ファイル | 内容 | 行数 |
|---------|------|------|
| `sales_data_202401.csv` | 2024年1月 売上データ | 約200行 |
| `sales_data_202402.csv` | 2024年2月 売上データ | 約180行 |
| `sales_data_202403.csv` | 2024年3月 売上データ | 約180行 |
| `employee_list.csv` | 社員名簿 | 15行 |
| `product_master.csv` | 商品マスター | 7行 |
| `merge_sample/*.csv` | 統合練習用 | 10ファイル |

**CSVの使い方:**
- Excelで直接開ける
- Pythonで `pd.read_csv()` で読み込める
- xlsx形式で保存し直すことも可能

### 生成スクリプト

| ファイル | 説明 |
|---------|------|
| `generate_csv.py` | CSVファイル生成（標準ライブラリのみ） |
| `generate_xlsx_colab.py` | xlsxファイル生成（Colab用） |

## データ内容

### 売上データ（sales_data_YYYYMM）

| 列名 | 内容 | 例 |
|------|------|-----|
| 日付 | 取引日 | 2024/01/15 |
| 部署 | 営業部、開発部、総務部、マーケティング部 | 営業部 |
| 担当者 | 担当者名 | 田中太郎 |

| 数量 | 販売数量 | 10 |
| 単価 | 商品単価 | 15000 |
| 金額 | 数量 × 単価 | 150000 |

### 社員名簿（employee_list）

| 列名 | 内容 | 例 |
|------|------|-----|
| 社員ID | E001形式 | E001 |
| 氏名 | 氏名 | 田中太郎 |
| 部署 | 所属部署 | 営業部 |
| 役職 | 部長、課長、主任、一般 | 部長 |
| 入社日 | 入社日 | 2015/04/01 |
| メール | メールアドレス | tanaka@example.com |

### 商品マスター（product_master）

| 列名 | 内容 | 例 |
|------|------|-----|
| 商品ID | P001形式 | P001 |
| 商品名 | 商品名 | 商品A |
| カテゴリ | 商品カテゴリ | カテゴリ1 |
| 単価 | 販売単価 | 5000 |
| 在庫数 | 現在の在庫数 | 100 |
| 備考 | 補足情報 | 主力商品 |

## GAS経験者へ

このサンプルデータは、GASでスプレッドシートを操作していた方が
Pythonでの処理を練習するためのものです。

**GASとの違い:**
- GAS: スプレッドシート（Google形式）をDriveから取得
- Python: Excel/CSV（ローカルファイル）を直接操作

**Pythonのメリット:**
- ローカルファイルを直接操作できる
- Googleドライブへのアップロード不要
- 100ファイルでも数秒で処理可能
