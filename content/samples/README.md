## CSVからExcelへの変換

### 方法1: Excelで開いて保存
1. CSVファイルをExcelで開く
2. 「名前を付けて保存」→「Excel ブック (.xlsx)」で保存

### 方法2: Pythonで変換
```python
import pandas as pd

# CSVを読み込んでExcelに保存
df = pd.read_csv('sales_data_202401.csv')
df.to_excel('sales_data_202401.xlsx', index=False)
```

### 方法3: 講座内のコードを実行
各章のコードにはサンプルデータを自動生成するコードが含まれています。
「Step 0」のコードを実行すれば、同等のデータが生成されます。

## GAS経験者へ

このサンプルデータは、GASでスプレッドシートを操作していた方が
Pythonでの処理を練習するためのものです。

GASとの違い:
- GAS: スプレッドシート（Google形式）
- Python: Excel/CSV（ローカルファイル）

Pythonなら、ローカルファイルを直接操作できます。
Googleドライブへのアップロードは不要です。
