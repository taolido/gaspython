# 挫折しない・壊さないPython入門
## ─ AIと一緒に学ぶ、GASの次のステップ

GAS講座の成功体験をベースに、Python講座を企画するためのドキュメント集です。

## 📌 コンセプト

**キャッチコピー:**
> 環境構築で詰まらない。依存関係で壊さない。AIが隣にいるから怖くない。

**4本柱:**
1. **環境構築で挫折させない** - Google Colabで即開始
2. **AIを学習パートナーにする** - エラー対処、理解促進のメソッド化
3. **目に見える成果を早期に出す** - 第3章で実用ツール完成
4. **環境を壊さない作法を教える** - venv必須、依存関係の基礎知識
